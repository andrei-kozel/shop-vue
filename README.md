This is mu simple Shop project written with Vue.js (Vuex, Vue Router). Just for practice purpose.

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
